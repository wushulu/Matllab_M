# 01 - Clarke Transform Module

## 简介 Introduction
本文件夹包含电机控制中基础的 Clarke 变换 (Clarke Transform) 的 Simulink 模型测试。Clarke 变换用于将三相静止坐标系 ($I_a, I_b, I_c$) 转换为两相静止坐标系 ($I_\alpha, I_\beta$)。

## 文件列表 File List
-   **`clarck.slx`**: Clarke 变换的 Simulink 仿真模型。
-   **`STM32CubeMXClark`**: 相关 STM32CubeMX 工程配置文件。
