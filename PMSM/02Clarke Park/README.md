# 02Clarke Park - Coordinate Transformations

## 简介 Introduction
本模块专注于坐标变换 (Coordinate Transformation)，这是 Field-Oriented Control (FOC) 的核心数学基础。

主要包含：
-   **Clarke 变换**: 3相静止 -> 2相静止
-   **Park 变换**: 2相静止 -> 2相旋转 ($d-q$ 轴)
-   及其相应的逆变换 (Inverse Clarke/Park)。
